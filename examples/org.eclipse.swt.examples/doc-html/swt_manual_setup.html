Standard Widget Toolkit (SWT) - Standalone Examples Manual Setup

Setup

There are several steps required before the SWT standalone examples can be run.
  • Copy the swt runtime library on the system path [System.getProperty("java.library.path:)].  On Linux, this is LD_LIBRARY_PATH.  (Is  LD_LIBRARY_PATH applicable to all shells?).
      • The swt runtime library can be found in the following directory, depending one which Operating System you are using to run Eclipse:
      • Windows - <install-dir>\eclipse\plugins\org.eclipse.swt\ws\win32\
      • Motif - <install-dir>/eclipse/plugins/org.eclipse.swt/ws/motif/
  • Create a Java Project to contain the examples.  Call it "SWT Examples".
  • Make sure the swt.jar file is on the java classpath.

    • Select the SWT Examples project and bring up the Properties dialog.  On the Java Build Path page, select the Libraries tab, click on the Add External JARs button and find swt.jar.  The swt.jar file can be found in the following directory, depending one which Operating System you are using to run Eclipse:
    • Windows - <install-dir>\eclipse\plugins\org.eclipse.swt\

      • Motif - <install-dir>/eclipse/plugins/org.eclipse.swt/
  • Make sure the swtexamples.jar is on the java classpath.

    • Select the SWT Examples project and bring up the Properties dialog.  On the Java Build Path page, select the Libraries tab, click on the Add External JARs button and find swtexamples.jar.  The swtexamples.jar file can be found in the following directory, depending one which Operating System you are using to run Eclipse:
    • Windows - <install-dir>\eclipse\plugins\org.eclipse.swt.examples

      • Motif - <install-dir>/eclipse/plugins/org.eclipse.swt.examples/

Running the Examples

In the Packages view of the Java Perspective, expand swtexamples.jar.  You will see all of the packages and other resource files present in the jar file.

All of the standalone SWT examples are intended to be run as a Java Application.  No parameters need be passed to any of these examples in order to run the example.  To run one of the example classes, simply expand the containing package of the runnable class, select the executable class, then click on the Run button on the toolbar.

Consult the documentation of each individual example for the names of its executable class(es) and additional details.

Package Manifest

swtexamples.jar contains the following examples:

  • Address Book
  • File Viewer
  • Hello World [1-5]
  • Hover Help
  • Image Analyzer
  • Java Viewer
  • Text Editor
